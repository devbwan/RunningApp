## 🧪 테스트 방법

### 웹 테스트
1. `npm run web` 실행
2. 브라우저에서 http://localhost:8081 접속
3. 탭 네비게이션 동작 확인
4. 각 화면 UI 렌더링 확인

### 모바일 테스트 (권장)
1. `npm run android` 또는 `npm run ios` 실행
2. GPS 권한 허용
3. 러닝 시작 → 경로 추적 → 종료 → 저장 테스트
4. 메달 획득 테스트

### 주요 테스트 시나리오
1. 러닝 시작 → 100m 이상 이동 → 종료 → 저장
2. 저장된 세션 목록 확인
3. 세션 상세 화면 확인
4. 통계 업데이트 확인
5. 메달 획득 확인
