## 🧪 테스트 방법

### 웹 테스트
1. 환경 변수 설정: `.env.web` 또는 `.env.local` 파일 생성
2. `npm run web:setup` 또는 `npm run web:local` 실행
3. 브라우저에서 http://localhost:8081 접속
4. 탭 네비게이션 동작 확인
5. 각 화면 UI 렌더링 확인
6. Google 로그인 테스트 (Firebase 설정 필요)

### 모바일 테스트 (권장)
1. 환경 변수 설정: `.env.android` 파일 생성
2. `npm run android:setup` 실행
3. GPS 권한 허용
4. 러닝 시작 → 경로 추적 → 종료 → 저장 테스트
5. 메달 획득 테스트
6. 인증 기능 테스트

### 주요 테스트 시나리오
1. **러닝 테스트**
   - 러닝 시작 → 50m 이상 이동 → 종료 → 저장
   - 저장된 세션 목록 확인
   - 세션 상세 화면 확인
   - 통계 업데이트 확인
   - 메달 획득 확인

2. **인증 테스트**
   - 게스트 모드로 앱 사용
   - Google 로그인 테스트
   - 로그아웃 후 새로고침 시 게스트 모드 확인

3. **웹 배포 테스트**
   - `npm run deploy` 실행
   - https://devbwan.github.io/ 접속
   - 모든 기능 정상 작동 확인
