## 🧪 테스트 가이드

### 웹 환경 테스트
```bash
npm run web
```
- 브라우저에서 http://localhost:8081 접속
- 탭 네비게이션 동작 확인
- 각 화면 UI 렌더링 확인
- ⚠️ GPS 기능은 웹에서 제한적

### 모바일 환경 테스트 (권장)
```bash
# Android
npm run android

# iOS  
npm run ios
```

### 테스트 시나리오
1. 러닝 시작 → GPS 권한 허용 → "러닝 시작"
2. 러닝 중 → 거리/시간/페이스 확인 → 일시정지/재개 테스트
3. 러닝 종료 → 50m 이상 이동 → 저장 확인
4. 기록 확인 → 기록 탭에서 저장된 세션 확인
5. 세션 상세 → 세션 카드 클릭 → 상세 화면 확인
6. 통계 확인 → 홈 화면에서 오늘의 기록 확인
7. 메달 확인 → 프로필에서 메달 진행도 확인
